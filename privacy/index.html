Privacy notes

Last updated 3 July 2026 · Complies with the New Zealand Privacy Act 2020

This site exists to teach. It does not exist to harvest you. What follows is the plain-language version of what we do and don't do with your data. If you want the legal-flavoured version, contact us — details below.

Short version. No analytics. No cookies from us. No IP tracking. No email list running through this site. Comments are hosted on GitHub Discussions via Giscus (MIT-licensed, open source) — not on this site. GitHub Pages hosts the files and keeps its own minimal server logs — that's out of our hands.

What we collect

Nothing directly. This site is a static site hosted on GitHub Pages. We have not installed Google Analytics, Facebook Pixel, or any other tracking script. We do not run a mailing-list widget on this site (the Substack newsletter lives at kiwidialectic.substack.com — its privacy terms apply there, not here).

Comments — Giscus / GitHub Discussions

Each page has an open comment thread powered by Giscus, a free and open-source (MIT) comment widget that maps every page on this site to a matching thread in GitHub Discussions on the site's public repo.

  • Comments are stored inside GitHub Discussions on the te-pa-course repo, not on this site. Anyone can read them without an account.
  • To post, you need a free GitHub account. There is no anonymous / guest posting — this is Giscus's one real trade-off, but it means everything you post is tied to your GitHub handle, publicly and permanently.
  • Whatever you type is public. It lives in the repo's discussion threads and can be quoted, indexed, and archived anywhere on the open web. Do not post anything you would not want quoted back to you.
  • To delete a comment you posted, use GitHub's own delete button on the discussion, or contact us (details below).
  • GitHub's privacy policy governs storage of comment data and account metadata. See GitHub's privacy statement.
  • The Giscus widget itself sets no tracking cookies, but the giscus.app script and the embedded GitHub iframe both load third-party JavaScript from those origins.

Server logs (GitHub Pages)

GitHub, which hosts these pages, keeps standard web-server access logs — including IP addresses — for a short period, primarily for abuse prevention. We do not access, request, or store those logs. See GitHub's privacy statement.

Fonts and third-party assets

  • Fonts (Bebas Neue, Inter) load from Google Fonts. Google may log your IP when your browser requests them. See Google's privacy policy.
  • The Giscus widget script loads from giscus.app and embeds an iframe served from GitHub.
  • External links to Substack, Ko-fi, GitHub, YouTube, or elsewhere are governed by those services' own privacy policies once you leave this site.

Your rights under the Privacy Act 2020

Under the New Zealand Privacy Act 2020 you have the right to ask what personal information (if any) we hold about you, to correct it, and to have it deleted. Because we do not run our own database, in practice this usually means asking us to delete a comment you left. Contact robertmorrisportraits@gmail.com.

Kids under 16

This site is aimed at adults. Nothing here is designed to collect information from kids under 16. If you are a parent or caregiver and want a comment deleted, email us and we'll sort it.

Changes

If we change how comments or hosting work, we'll update this page and note the new date at the top.
